# Pipeful AI

The Voice-First CRM Assistant - Stop Typing. Start Closing.

## Overview

Pipeful AI is an intelligent voice assistant that helps sales professionals update their CRM (Salesforce & HubSpot) through natural conversation. Update deals, log activities, and keep your pipeline pristine—all while you're away from your desk.

## Features

- **Voice-First CRM Updates**: Update Salesforce and HubSpot through natural conversation
- **Intelligent Interviewer**: AI asks critical questions to ensure complete and accurate pipeline data
- **Methodology Support**: Works with MEDDPICC, BANT, SPIN, and custom methodologies
- **Real-Time Sync**: All updates sync instantly to your CRM
- **Activity Logging**: Automatically log calls, emails, and meetings

## Website Structure

- `index.html` - Main landing page
- `pricing.html` - Pricing page with three tiers
- `about-us.html` - Company information
- `careers.html` - Job openings and company culture
- `contact.html` - Contact form and information
- `salesforce-integration.html` - Salesforce integration details
- `hubspot-integration.html` - HubSpot integration details
- `security.html` - Security and privacy information

## Setup

1. Clone this repository
2. Open `index.html` in a web browser
3. All pages are static HTML and can be hosted on any web server

## Waitlist Form

The waitlist form is powered by Fillout. The form embed is configured in `index.html` and opens in a modal when users click "Join our Waitlist".
